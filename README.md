## ‚ö° Overview

When disasters strike, milliseconds matter. **CrisisLink** is a modern, high-performance command center dashboard built to solve the logistical nightmares of global crisis response. By leveraging an interactive mapping system, real-time intelligence feeds, and an integrated Logistics AI, CrisisLink empowers field commanders to make split-second, data-driven decisions that save lives.

## ‚ú® Features

- **Global Command Grid:** An interactive, CSS Grid-based map projection with dynamic scanning effects, visualizing active dispatch routes and critical threat nodes across all sectors.
- **Live Intel Feed:** A real-time, scrolling multimodality dashboard that ingests critical alerts, warnings, and dispatch logs with automated mitigation strategies.
- **Logistics AI Chat:** Powered by the Vercel AI SDK, our system analyzes incoming distress text scenarios (e.g., *"We need 500 blankets in Sector 3"*) and outputs optimal logistical routing, inventory checks, and deployment strategies.
- **Series-A Aesthetic:** Built with a breathtaking, immersive dark-mode UI utilizing Tailwind CSS, refined gradients, glassmorphism, and smooth micro-animations.

## üöÄ Getting Started

Follow these steps to run CrisisLink locally on your machine.

### Prerequisites

- Node.js (v18.0.0 or higher)
- npm, yarn, pnpm, or bun

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/aaditx/CrisisLink.git
   cd CrisisLink
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## üìç Navigating the App

1. **Landing Page (`/`):** Experience the premium hero section and click "Launch Dashboard" to enter the command center.
2. **Control Center (`/dashboard`):** 
   - View top-level KPIs (Crises, Personnel, Supplies).
   - Observe the MapView node grid parsing threat levels.
   - Read the incoming live intel logs.
   - Interact with the AI Chat card by submitting a crisis scenario.
